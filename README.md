# Code examples for Movie Finder

## Setup

You can install all the packages for all the projects up front, saving you time in the future. To do so, from this directory:

```
npm i

Unless you have a quantum computer connected directly to an Amazon data center, this task will take a some time to complete.
```

## Running code

You can run local server and check the app locally. To do so, from this directory:

```
npm run serve
```

## Running tests

You run some test to check if the app goes properly. To do so, from this directory:

```
npm run test
```
